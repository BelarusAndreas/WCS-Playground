### Features
-   Keine Backlinks
-   Zwei vorgegebene Währungen
-   Benutzer kann andere Werbungen auswählen
-   Leicht und schnell einzubinden
-   Per CSS an eigenen Style anpassbar

### Quellcode:

```html
<experte comparison="currency" from="EUR" to="BYN"></experte>
<script src="https://partner.experte.de/vergleich.js"></script>
```

### Benutzung:
1.  Standardwährungen  from="EUR"  und  to="BYN"  anpassen.  
    Hier gibt es eine Liste mit allen Währungskürzel:  [iban.com (deutsch)](https://de.iban.com/currency-codes)
2.  Style nach eigenen Wünschen anpassen  
    Dazu per CSS die Box per  data-box-identifier="com.woltlab.wcf.genericBoxXXX"  ansprechend und nach eigenen belieben anpassen.
